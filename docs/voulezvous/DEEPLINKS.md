# Deep Links (padrão)

- `omni://room/{room_id}` — abre Party/Circle/Stage conforme tipo do room
- `omni://profile/{user_id}` — abre Profile Sheet + Action Bar
- `omni://invite/{invite_id}` — resolve Invite → gate → entra na room
- `https://voulezvous.tv/@{user}` — Stage do criador (Offline | Live | Replay)

Navegação sempre **sem recarregar o player** (VideoShell persistente).
